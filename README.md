# camp-website
How to run
Must have: 
  - sbt newest
  - java newest
  - nodejs
1. Script(Update soon)

  'cd camp-website'
  
  'sh ./start.sh'
  
2. Manualy

   'cd camp-website/project/backend'
   
   'sbt run'
   
   
  Open new command line
  
   'cd camp-website/project/frontend'
   
   'npm install'
   
   'npm start'
   
 View your website on address: localhost:3000
